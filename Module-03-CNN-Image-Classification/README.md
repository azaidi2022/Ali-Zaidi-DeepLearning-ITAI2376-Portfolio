# Module 03 - CNN Image Classification

## Overview
This lab focuses on convolutional neural networks (CNNs) and their applications in image classification tasks. The project explores how CNN architectures process visual data using convolutional layers, pooling operations, and feature extraction techniques.

## Objectives
- Understand the structure and function of CNNs
- Explore image preprocessing techniques
- Train and evaluate image classification models
- Analyze model accuracy and performance

## Technologies Used
- Python
- Jupyter Notebook
- TensorFlow / Keras
- NumPy
- Matplotlib

## Files Included
- `Module_03_Lab_AliZaidi(2).ipynb` — Main notebook containing implementation, outputs, and analysis

## Key Learning Outcomes
- Developed practical understanding of convolutional neural networks
- Learned how CNNs extract features from images
- Explored deep learning workflows for computer vision tasks
