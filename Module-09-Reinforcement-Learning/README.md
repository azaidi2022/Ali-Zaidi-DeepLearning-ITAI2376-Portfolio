This lab introduces reinforcement learning concepts and explores how intelligent agents learn through interaction with environments. The project focuses on reward systems, decision-making processes, and agent-based learning strategies used in modern AI systems.

- Analyze rewards, policies, and decision-making
- Evaluate learning behavior over time
